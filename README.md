# Personal Portfolio Website

**Portfolio website built with PHP, JavaScript, and Bootstrap.**

This repository showcases my personal portfolio, highlighting my projects, skills, and a contact system. The website is designed to demonstrate my abilities in web development, front-end and back-end programming, and responsive design.

---

## Features

- **Home Section:** Introduction and overview of my skills.
- **Projects Section:** Displays multiple projects with descriptions and links.
- **Contact Form:** Dynamic contact form built with PHP for email submissions. *(Runs locally with XAMPP/WAMP)*.
- **Responsive Design:** Fully responsive layout using Bootstrap.
- **Modern UI:** Clean and professional interface for user-friendly navigation.

---

## Technologies Used

- **Front-end:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Back-end:** PHP
- **Database:** MySQL (for contact form storage)
- **Version Control:** Git and GitHub
